# TESCII

An open-source text to ascii art convertor made in rust. 

### 💻 Techs:

**Language:** Rust

**Libs:** Figlet-Rs and ClearScreen

### 📜 License:

This project is licensed under [MIT license](LICENSE)

### 🛠️ How to convert to a command:

Step by step:

  1. Go to the project folder using your terminal
  2. Use `cargo install --path .`
  3. Now, just use `tescii` and write your ascii art texts!
